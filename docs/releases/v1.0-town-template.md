# Town Warden v1.0 — Cloneable Civic Intelligence Template

Town Warden v1.0 packages the project as a cloneable civic-intelligence
template for UK towns. The default example remains Blackpool, but developers can
adapt the dashboard by editing town configuration rather than rewriting the
backend or frontend.

Town Warden is an experimental civic-tech prototype. It is not official council,
police, NHS, or emergency-service advice.

## Highlights

- FastAPI backend
- Static HTML/CSS/JavaScript frontend
- PostgreSQL production support
- SQLite fallback for local development and tests
- Config-driven town setup through `TOWN_CONFIG`
- Blackpool default config
- Police.uk public incident intelligence
- Open-Meteo weather context
- Street Manager street works webhooks
- Source health and runtime status
- Analytics
- History
- Map data with coordinate confidence
- Source-aware agent insights
- Protected admin/developer routes
- Backend tests
- Public disclaimers marking the dashboard as experimental and not official public authority advice

## Template Use

Developers can copy `config/towns/blackpool.json`, edit the town name, map
centre, zones, bounding box, authority keywords, source labels, and dashboard
wording, then point `TOWN_CONFIG` at the new JSON file.

See `docs/ADAPT_FOR_YOUR_TOWN.md` for the full adaptation workflow.

## Data Limitations

- Police.uk location data is approximate/anonymised.
- Street Manager filtering depends on available payload fields and configured
  local geography.
- Weather context is not an official warning feed.
- Agent insights are rule-based and source-limited.
- Town Warden should not be used as an emergency, dispatch, council, police,
  NHS, or public authority system.
